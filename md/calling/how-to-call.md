สรุปกระบวนการ Call Center พรรคประชาชน (ทีมจังหวัด)
โครงสร้างการโทร (3 ส่วนหลัก)

## ส่วนที่ 1 — บันทึกสถานะการติดต่อ
ก่อนโทร : เช็คข้อมูลอ่านประวัติการโทรให้เรียบร้อยเพื่อสร้างบทสนทนา

## ส่วนที่ 2 — สำรวจการมีส่วนร่วม (8 คำถาม)

เปิดสาย : สวัสดีครับ/ค่ะ ขอสาย [ชื่อ] หน่อยได้ไหมครับ/ค่ะ — โทรจากทีมพรรคประชาชนราชบุรีครับ/ค่ะ ตอนนี้สะดวกคุยสักครู่ไหมครับ/ค่ะ?

1. ถิ่นที่อยู่ : ปกติอยู่ที่ราชบุรีเป็นหลักเลยไหมครับ/ค่ะ หรือทำงานที่อื่น?
2. ระดับความสนใจ : ที่สมัครสมาชิกไว้ ตอนนี้อยากมีส่วนร่วมกับพรรคแค่ไหนครับ/ค่ะ — แค่สนับสนุน / ติดตามข่าว / ร่วมกิจกรรม / หรืออยากช่วยงานเลย?
3. อาสาสมัคร : สนใจมาช่วยงานทีมราชบุรีบ้างไหมครับ/ค่ะ?
4. วัน–เวลา (ถ้าสนใจร่วมกิจกรรม) : สะดวกวันธรรมดา หรือเสาร์–อาทิตย์ครับ/ค่ะ? แล้วช่วงเช้า บ่าย หรือเย็น?
5. ประเภทกิจกรรม : ชอบกิจกรรมแบบไหนบ้างครับ/ค่ะ เช่น เสวนา / จิตอาสา / ออนไลน์ / รณรงค์?
6. ช่องทางข่าวสาร : รับข่าวสารสะดวกผ่านช่องทางไหนครับ/ค่ะ — Line / Facebook / SMS?
7. ทีมจังหวัด : ทราบไหมครับ/ค่ะว่าราชบุรีมีทีมพรรคทำงานอยู่? (ถ้าไม่รู้ → แนะนำสั้นๆ → ถามว่าสนใจร่วมกิจกรรมไหม → ชวนเป็นอาสา)

ปิดสาย : ขอบคุณมากนะครับ/ค่ะ ติดต่อทีมราชบุรีได้ที่ LINEOA เดี๋ยวทางเราจะจัดส่งให้ทาง SMS หลังจากวางสายครับ/ค่ะ สวัสดีครับ/ค่ะ

## ส่วนที่ 3 — ต่ออายุสมาชิก

1. เปิดสาย : สวัสดีครับ/ค่ะ ขอสาย [ชื่อ] — โทรจากทีมพรรคประชาชนราชบุรีครับ/ค่ะ สะดวกคุยสักครู่ไหมครับ/ค่ะ?
2. แจ้งสถานะ : สมาชิกของคุณ [ชื่อ] จะครบอายุวันที่ [X] นะครับ/ค่ะ
3. ชวนต่ออายุ : ต่ออายุได้สองแบบครับ/ค่ะ รายปี 50 บาท (ไม่มีบัตร) หรือ 150 บาท (รับบัตรด้วย) แนะนำตลอดชีพ เริ่มต้น 500 บาทขึ้นไป
4. แจ้งช่องทาง : ต่ออายุผ่านเว็บพรรค หรือทักไลน์ @mfp_ratchaburi ได้เลยครับ/ค่ะ หลังวางสายจะส่ง SMS ลิงก์ให้ด้วย
5. สิทธิ์เพิ่มเติม : ในอนาคตสมาชิกจะสามารถใช้สิทธิ์เลือกตั้งตัวแทนพรรคออนไลน์ได้ด้วยนะครับ/ค่ะ
6. ปิดสาย : ขอบคุณมากนะครับ/ค่ะ ถ้ามีคำถามเพิ่มเติมทักไลน์มาได้เลยนะครับ/ค่ะ สวัสดีครับ/ค่ะ
