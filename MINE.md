สรุปขั้นตอนทั้งหมดครับ:

1. ติดตั้ง Meilisearch


curl -L https://install.meilisearch.com | sh
sudo mv meilisearch /usr/local/bin/
2. ตั้ง systemd service


sudo nano /etc/systemd/system/meilisearch.service

[Unit]
Description=Meilisearch
After=network.target

[Service]
ExecStart=/usr/local/bin/meilisearch --master-key=YOUR_KEY --db-path /var/lib/meilisearch
Restart=always
User=root

[Install]
WantedBy=multi-user.target

sudo systemctl enable --now meilisearch
3. เพิ่มใน .env


MEILISEARCH_HOST=http://localhost:7700
MEILISEARCH_KEY=YOUR_KEY
4. รัน SQL migration


mysql -u pple_dcbot -p pple_volunteers < scripts/migration-forum.sql
5. ติดตั้ง npm package


npm install meilisearch
6. Deploy commands


node deploy-commands.js
7. Restart bot


pm2 restart bot  # หรือ process ที่ใช้อยู่
8. Backfill


node scripts/backfill-forum.js --guild GUILD_ID
9. Setup dashboard (ทำใน Discord)

/panel forum channel:#ช่องที่ต้องการ ทีละช่องที่อยากมี dashboard
